# California House Price Prediction

A Machine Learning based web application that predicts house prices using Random Forest Regression.

## Features

- User input based prediction
- Flask web dashboard
- Interactive charts
- Location visualization
- ML pipeline preprocessing

## Technologies

- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS/JavaScript

## Run Project

Install dependencies:

pip install -r requirements.txt

Run:

python app.py
